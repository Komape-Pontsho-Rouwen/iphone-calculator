About This Project

This is a fully functional web-based calculator app , built to mimic the design and behaviour of the native iOS calculator.Built with clean semantic HTML , modern CSS with a bit of animation and & vanilla JS for calculations & interaction.

This project demonstrates proficiency in front-end development , the ability to recreate complex interfaces and attention to details in UI/UX design.
